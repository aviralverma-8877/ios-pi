feat: transform into GNOME-like desktop environment

Completely redesigned the desktop from a fullscreen app to a proper
GNOME-like desktop environment with panels and window management.

**Major Changes:**

Desktop Shell Architecture:
- Changed from single fullscreen window to proper desktop shell
- Added top panel (like GNOME's top bar) with:
  • Activities button to open application launcher
  • Application title in center
  • System indicators (temperature)
  • Clock/time display
  • Power menu button

- Added desktop background area that stays behind windows
- Added dock at bottom (macOS/Ubuntu style) with favorite apps
- Applications now open as separate windows, not fullscreen

Application Launcher:
- Created Activities overlay (like GNOME Activities)
- Search functionality to filter applications
- Grid layout with all available apps
- Keyboard shortcuts (ESC to close)

Window Management:
- File Manager opens as standalone window
- Terminal opens as standalone window
- Settings opens as standalone window
- Each window can be moved, resized independently
- Proper window stacking and focus management

User Experience:
- More like traditional desktop environment
- Apps don't take over entire screen
- Can have multiple windows open simultaneously
- Better workflow for multitasking
- Familiar GNOME-like interaction patterns

Visual Improvements:
- Nord color scheme for terminal
- Better styling for all panels
- Hover effects on buttons
- Smooth transitions
- Professional desktop appearance

Technical Details:
- Desktop uses Qt.WindowStaysOnBottomHint to stay behind
- Windows use Qt.Window flag for proper window management
- Panel is frameless but always on top
- Dock has transparency and rounded corners
- Application launcher is modal dialog overlay

Breaking Changes:
- No longer boots into fullscreen app mode
- Now provides proper desktop environment
- Home button removed (Activities button instead)
- Different navigation pattern (panel + dock vs single screen)

Benefits:
✓ More professional desktop experience
✓ Better window management
✓ Familiar to GNOME users
✓ Can run multiple apps simultaneously
✓ Better for productivity
✓ Proper desktop environment, not just launcher
