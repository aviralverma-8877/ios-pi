🚀 READY TO DEPLOY

   On Raspberry Pi:

   1. git clone <repository-url>
   2. cd pi-ios-desktop/deb-build
   3. ./build-desktop-deb.sh
   4. cd .. && sudo dpkg -i pi-ios-desktop_1.0.0_all.deb
   5. sudo /usr/share/doc/pi-ios-desktop/set-default.sh
   6. sudo reboot

   Your Pi will boot into GNOME-like iOS-styled desktop!
