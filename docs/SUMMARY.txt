🖥️ DISPLAY CONFIGURATION (One-time setup):

   For 4.3" displays (800x480):
   
   $ sudo nano /boot/config.txt
   
   Add these lines:
   
   hdmi_group=2
   hdmi_mode=87
   hdmi_cvt=800 480 60 6 0 0 0
   gpu_mem=128
   disable_overscan=1
   
   Then reboot:
   $ sudo reboot

🆘 COMMON ISSUES:

   Desktop doesn't show at login
   → Check: ls -l /usr/share/xsessions/pi-ios-desktop.desktop
   → Run: sudo systemctl restart lightdm
   
   Black screen after login
   → Check: cat ~/.xsession-errors
   → Install: sudo apt-get install python3-pyqt5 openbox
   
   Touch not working
   → Install: sudo apt-get install xinput-calibrator
   → Calibrate: xinput_calibrator
   
   Can't shutdown/reboot
   → Edit: sudo visudo
   → Add: pi ALL=(ALL) NOPASSWD: /sbin/shutdown
   → Add: pi ALL=(ALL) NOPASSWD: /sbin/reboot
   
   📖 Full troubleshooting: TROUBLESHOOTING.md

💡 USAGE MODES:

   Auto-Boot Mode (Recommended):
   • Pi boots straight into iOS desktop every time
   • Run: sudo /usr/share/doc/pi-ios-desktop/set-default.sh
   
   Select at Login Mode:
   • Keep LXDE and choose at login screen
   • Don't run set-default.sh after install
   
   Manual Mode:
   • Run: pi-ios-desktop (when you want it)

🗑️ UNINSTALL:

   $ sudo apt-get remove pi-ios-desktop
   $ sudo reboot
   
   Or use script:
   $ chmod +x uninstall-desktop.sh
   $ sudo ./uninstall-desktop.sh
   
   Original LXDE desktop is automatically restored!
