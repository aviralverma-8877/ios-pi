   • Mobile app feel

🔧 TECHNICAL ARCHITECTURE

   Desktop Shell:
   • Qt.WindowStaysOnBottomHint for desktop
   • Qt.Window for application windows
   • Qt.Dialog for launcher overlay
   • OpenBox manages window decorations

   Window Management:
   • Tracks open windows in dictionary
   • Prevents duplicate windows
   • Raises existing windows
   • Proper focus management

   Styling:
   • Nord color scheme for terminal
   • iOS-inspired gradient background
   • Semi-transparent panels
   • Hover effects throughout

🚀 HOW TO USE

   1. Build & Install (same as before):
      cd deb-build
      ./build-desktop-deb.sh
      cd ..
      sudo dpkg -i pi-ios-desktop_1.0.0_all.deb

   2. New Interaction:
      • Click "Activities" → Opens app launcher
      • Search apps by name
      • Click dock icons for quick launch
      • Multiple windows stay open
      • Use window manager to move/resize

   3. Power Menu:
      • Click ⏻ button in top-right
      • Choose Logout/Reboot/Shutdown
